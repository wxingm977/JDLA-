# 1. numpy.c_
カンマ区切りで指定した配列を axis=-1 方向で結合する。  
```
a = np.array([[1, 2],  
              [3, 4]])  
b = np.array([[5, 6],  
              [8, 9]])  
```
axis=-1 (a.k.a axis=1) 方向で結合する。
```
c = np.c_[a, b]  
print(c)  
# [[1 2 5 6]  
#  [3 4 8 9]]  
```

```
c = np.array((1, 2, 3))  
d = np.array((4, 5, 6))
np.c_[c, d]  
array([[1, 4],  
       [2, 5],  
       [3, 6]]) 
       
>>> np.c_[np.array([[1,2,3]]), 0, 0, np.array([[4,5,6]])]  
array([[1, 2, 3, 0, 0, 4, 5, 6]])
```
